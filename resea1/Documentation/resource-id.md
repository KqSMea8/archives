Resource ID
-----------

- `@`: channel
- `+`: thread
- `#`: thread group

## Global
- `@<node>:thread_group:channel`
- `+<node>:thread_group:thread`
- `#<node>:thread_group`

## Node-local
- `@thread_group:channel`
- `+thread_group:thread`
- `#thread_group`

## Thread-group-local
- `@channel`
- `+thread`
