# 2fa
A command-line 2-factor authentication manager.

Register an account:
```
$ 2fa 'AWS (seiya)' '555555'
```

Generate codes:
```
$ 2fa
AWS (root)          281692      (23)
AWS (seiya)         628110      (23)
Dropbox             713351      (23)
GitHub              719876      (23)
Google              884966      (23)
```
