Supershow
==========

A handy markdown presentation tool.

Getting Started
---------------
```
$ npm install -g supershow
$ supershow file.md
```

Markdown
--------
```md
---
title: The title of presentation
---

# The title of presentation
Your name

----
# Next slide title
- list
  - nested list
    1. numbered
    2. list
```
