MakeStack

Releases Documentation GitHub MakeStack Cloud

A full-stack IoT Framework for super-rapid prototyping.

  • Intuitive Node.js API with plugins
  • Electron-based Desktop app
  • Minimialized (< 32MB) Linux distribution for Raspberry Pi
  • Rails-based backend server
  • Integrations with IFTTT, ThingSpeak, and Webhooks
  • Fully open sourced (MIT)
