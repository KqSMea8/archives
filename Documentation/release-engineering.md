Checklist
---------
**Don't forget to update `tools/publish`!

- [ ] Run `./tools/release`.

- [ ] `git tag`
- [ ] `git push --tags`
- [ ] Run `./tools/publish`.
- [ ] [Create a Release on GitHub](https://github.com/seiyanuta/makestack/releases/new).
