MakeStack

getting started docs device api server api code

A minimalistic IoT for super-rapid prototyping.

  • Write device-side with Arduino-compatible API.
  • Write server-side in Node.js with easy-to-use API.
  • Deploy to a serverless platform with one command.
  • Remote update, remote procedure call, and more.
  • Supports only ESP-32 for now.
  • Fully open sourced (CC0/MIT).
