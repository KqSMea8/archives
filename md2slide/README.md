Supershow creates a PDF presentation from Markdown by one command.

Getting Started
---------------
```
$ npm install -g supershow
$ supershow file.md -o file.pdf
```

Markdown
--------
```
---
title: The title of presentation
---

# The title of presentation
Your name

----
# Next slide title
- list
  - nested list
    1. numbered
    2. list
```
